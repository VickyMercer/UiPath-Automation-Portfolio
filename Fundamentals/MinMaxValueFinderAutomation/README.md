# MinMaxValueFinderAutomation

## Overview
This automation finds the minimum and maximum values in an array of numbers by iterating through the array and comparing each element.

## What it does
1. Initializes an array of integers (with an optional, currently disabled, input dialog available for entering custom numbers)
2. Sets initial minimum and maximum values to the first element of the array
3. Iterates through each number in the array
4. Checks and updates the minimum value if a smaller number is found
5. Checks and updates the maximum value if a larger number is found
6. Writes the final minimum value to the console
7. Writes the final maximum value to the console

## Workflow
- `Main.xaml`

## Output
- Console output displaying the minimum value found in the array
- Console output displaying the maximum value found in the array

## Key concepts practiced
- Array initialization and indexing
- ForEach iteration
- Conditional comparisons for tracking min/max
- Console output with WriteLine
