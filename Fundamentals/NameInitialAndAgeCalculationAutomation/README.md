# NameInitialAndAgeCalculationAutomation

## Overview
This automation collects a user's full name and birth year through input dialogs, then calculates and displays derived values — the first letter of their name and their current age.

## What it does
1. Prompts the user to enter their full name via an input dialog
2. Extracts the first letter of the entered name
3. Prompts the user to enter their birth year via an input dialog
4. Calculates the user's age based on the birth year
5. Displays a message showing the first letter and calculated age
6. Writes the first letter to the console
7. Logs the calculated age

## Workflow
- `Main.xaml`

## Output
- A message is displayed showing the extracted first letter and calculated age
- Console output of the first letter
- A log entry recording the calculated age

## Key concepts practiced
- Input dialogs for capturing user input
- String manipulation (substring extraction)
- Variable assignment and arithmetic operations
- Message boxes for user feedback
- Console output with WriteLine
- Logging with different log levels
