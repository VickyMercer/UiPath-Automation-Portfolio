# List Sort And Reverse Automation

## Overview
This automation builds a list of items, sorts it alphabetically, reverses the sorted order, and logs the results at each stage.

## What it does
1. Builds a list of string items
2. Sorts the list in alphabetical order
3. Logs the sorted list
4. Reverses the order of the sorted list
5. Logs the reversed list
6. Logs the first 3 items from the reversed list

## Workflow
- `Main.xaml`

## Output
- Log entries showing the sorted list, the reversed list, and the top 3 items from the reversed list

## Key concepts practiced
- Collection (List) building
- Invoking built-in .NET methods (Sort, Reverse) on a collection
- LINQ method usage (Take)
- Logging with different log levels

## Requirements
- UiPath Studio
