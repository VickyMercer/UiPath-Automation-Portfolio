# Array Name Update Automation

## Overview
This UiPath automation performs array indexing and string concatenation to 
generate and update a full name value.

## What it does
1. Initializes a string array with two elements
2. Concatenates the array elements and displays the result
3. Updates one array element with a new value
4. Concatenates the array again and displays the updated result

## Workflow
- **Main.xaml** — contains the full automation logic (array initialization → 
  concatenation and display → element update → re-concatenation and display)

## Output
- A message is displayed showing the initial concatenated value
- A second message is displayed showing the updated concatenated value

## Key concepts practiced
- Array/collection initialization and indexing
- String concatenation
- Array element updates
- Sequential output messaging

## Requirements
- UiPath Studio
- UiPath.System.Activities package
